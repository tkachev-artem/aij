# AIJ – веб-сервис для анализа техники бега по загруженному видео

Моя fullstack разработка, фронт – vite, react, typescript, zustand, react query, sass; бэкенд — bun, express, typescript, zod, также используется turborepo. 
Также в проекте используется ml-сервис, который разработан моим коллегой, на базе sam3d и yolo, но не опубликован в открытом доступе.

### Команды для старта

```bash
bun install
```
